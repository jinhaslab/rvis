이 책에 대하여

이 책은 보건학, 역학, 임상 통계 분야의 대학원생과 전문가를 위한 R 시각화 실습 가이드입니다. ggplot2의 “그래픽 문법(Grammar of Graphics)”을 기반으로, 데이터 탐색부터 학술지 출판, 대화형 대시보드 구축까지 전 과정을 다룹니다.

0.0.1 📂 1단계: RStudio 프로젝트 생성

먼저 프로젝트를 만드세요!

  1. RStudio 열기
  2. File → New Project → New Directory → New Project
  3. 프로젝트 이름: rvis-practice
  4. Create Project 클릭

0.0.2 📥 2단계: 실습 스크립트 다운로드

프로젝트 안에서 스크립트를 다운로드하세요.

방법 1: GitHub에서 직접 다운로드

  1. GitHub 저장소 접속
  2. code 폴더 클릭
  3. setup.R 클릭 → Raw 버튼 → 우클릭 “다른 이름으로 저장”
  4. 프로젝트의 code/ 폴더에 저장 (폴더가 없으면 생성)
  5. data-simulation.R도 동일하게 저장

직접 링크: - setup.R 다운로드 - data-simulation.R 다운로드

방법 2: R Console에서 자동 다운로드 (더 쉬움!)

# RStudio Console에서 실행 (프로젝트 내에서)
# 폴더 자동 생성 + 스크립트 다운로드
dir.create("code", showWarnings = FALSE)
dir.create("data", showWarnings = FALSE)
dir.create("data/processed", showWarnings = FALSE)

download.file(
  "https://raw.githubusercontent.com/jinhaslab/rvis/main/code/setup.R",
  "code/setup.R"
)

download.file(
  "https://raw.githubusercontent.com/jinhaslab/rvis/main/code/data-simulation.R",
  "code/data-simulation.R"
)

# 확인
list.files("code")  # "setup.R" "data-simulation.R" 보여야 함

0.4.2 2. 필수 패키지 설치

모든 필요한 패키지는 code/setup.R에서 자동으로 설치됩니다:

# 프로젝트 루트에서 실행

주요 패키지:

  • Core: tidyverse, here, ggplot2
  • Epidemiology: incidence2, surveil
  • Spatial: sf, tmap
  • Clinical: survival, ggsurvfit, metafor
  • Interactive: plotly, shiny
  • Publication: patchwork, ggrepel, ggpubr
패키지 설치 시간

처음 실행 시 모든 패키지를 설치하는 데 10-20분 정도 소요됩니다. 인터넷 연결을 확인하고 충분한 시간을 두고 실행하세요.

0.4.3 3. 실습 데이터 준비

방법 1: 시뮬레이션 데이터 생성 (권장)

# 실습용 데이터 자동 생성
source("code/data-simulation.R")

다음 파일이 data/processed/에 생성됩니다:

  • health_survey.csv - 건강검진 데이터 (N=1,000)
  • disease_incidence.csv - 월별 질병 발생률
  • regional_disease.csv - 시도별 질병 데이터
  • clinical_trial.csv - 임상시험 데이터
  • meta_analysis.csv - 메타분석 데이터
  • covid_timeseries.csv - COVID-19 유사 시계열

방법 2: 패키지 내장 데이터 사용

별도 파일 없이 R 패키지에 내장된 데이터를 직접 사용합니다:

# 예시
data(mtcars)      # 기본 R
data(iris)        # 기본 R
library(survival)
data(lung)        # 생존 분석

library(outbreaks)
data(ebola_sim)   # 에볼라 유행 데이터
데이터 전략

이 책의 모든 실습은 재현 가능성을 최우선으로 합니다:

  1. ✅ 패키지 내장 데이터 우선 사용
  2. ✅ 시뮬레이션 데이터 (재현 가능한 set.seed())
  3. ❌ 외부 다운로드 최소화 (Shapefile만 예외)

따라서 인터넷이 없는 환경에서도 대부분의 실습이 가능합니다.

## 💻 실습 환경 준비

### 1. R과 RStudio 설치

```bash
# R 다운로드: https://cran.r-project.org/
# RStudio 다운로드: https://posit.co/download/rstudio-desktop/
```

**권장 버전:**

- R ≥ 4.3.0
- RStudio ≥ 2023.06.0

### 2. 필수 패키지 설치

모든 필요한 패키지는 `code/setup.R`에서 자동으로 설치됩니다:

```{r}
#| eval: false
#| echo: true

# 프로젝트 루트에서 실행
source("code/setup.R")
```

주요 패키지:

- **Core**: `tidyverse`, `here`, `ggplot2`
- **Epidemiology**: `incidence2`, `surveil`
- **Spatial**: `sf`, `tmap`
- **Clinical**: `survival`, `ggsurvfit`, `metafor`
- **Interactive**: `plotly`, `shiny`
- **Publication**: `patchwork`, `ggrepel`, `ggpubr`

::: {.callout-tip}
## 패키지 설치 시간

처음 실행 시 모든 패키지를 설치하는 데 **10-20분** 정도 소요됩니다.
인터넷 연결을 확인하고 충분한 시간을 두고 실행하세요.

### 3. 실습 데이터 준비

#### 방법 1: 시뮬레이션 데이터 생성 (권장)

```{r}
#| eval: false
#| echo: true

# 실습용 데이터 자동 생성
source("code/data-simulation.R")
```

다음 파일이 `data/processed/`에 생성됩니다:

- `health_survey.csv` - 건강검진 데이터 (N=1,000)
- `disease_incidence.csv` - 월별 질병 발생률
- `regional_disease.csv` - 시도별 질병 데이터
- `clinical_trial.csv` - 임상시험 데이터
- `meta_analysis.csv` - 메타분석 데이터
- `covid_timeseries.csv` - COVID-19 유사 시계열

#### 방법 2: 패키지 내장 데이터 사용

별도 파일 없이 R 패키지에 내장된 데이터를 직접 사용합니다:

```{r}
#| eval: false
#| echo: true

# 예시
data(mtcars)      # 기본 R
data(iris)        # 기본 R
library(survival)
data(lung)        # 생존 분석

library(outbreaks)
data(ebola_sim)   # 에볼라 유행 데이터
```

::: {.callout-note}
## 데이터 전략
