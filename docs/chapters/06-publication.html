8.1 6.1 가독성 향상: ggrepel

(내용 추가 예정 - PDF 페이지 18)

8.1.1 6.1.1 텍스트 라벨 겹침 문제

library(ggrepel)

# 산점도에 라벨 추가 (겹치지 않게)
ggplot(mtcars, aes(x = wt, y = mpg, label = rownames(mtcars))) +
  geom_point(color = 'red') +
  geom_text_repel() +  # geom_text() 대신
  theme_classic(base_size = 16)

8.2 6.2 다중 플롯 조합: patchwork

(내용 추가 예정 - PDF 페이지 19)

8.2.1 6.2.1 patchwork의 혁신

library(patchwork)

# 여러 플롯 생성
p1 <- ggplot(mtcars, aes(mpg, disp)) + geom_point()
p2 <- ggplot(mtcars, aes(gear, disp, group = gear)) + geom_boxplot()
p3 <- ggplot(mtcars, aes(disp, qsec)) + geom_smooth()

# 조합: (p1과 p2를 나란히) / 그 아래 p3
(p1 | p2) / p3 +
  plot_annotation(tag_levels = 'A')  # A), B), C) 태그

    label = "p.signif"  # *, **, *** 표시
  ) +
  stat_compare_means(label.y = 50)  # 전체 그룹 비교 p-value

```{r}
#| eval: false
#| echo: true

library(ggrepel)
```

## 6.2 다중 플롯 조합: patchwork

(내용 추가 예정 - PDF 페이지 19)

### 6.2.1 patchwork의 혁신

```{r}
#| eval: false
#| echo: true

library(patchwork)

# 여러 플롯 생성
p1 <- ggplot(mtcars, aes(mpg, disp)) + geom_point()
p2 <- ggplot(mtcars, aes(gear, disp, group = gear)) + geom_boxplot()
```
